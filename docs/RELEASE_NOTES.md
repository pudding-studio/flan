## v1.0.4

### 새로운 기능
- **Flan Agent 채팅** — 대화형으로 캐릭터 생성·수정·편집 가능
- **커스텀 제조사(Provider) 그룹핑** — 제조사별로 URL/Key/API포맷을 공유하며 하위 모델 여러 개 등록
- **제조사 드롭다운에 커스텀 제조사 표시** — "커스텀" 단일 항목 대신 등록한 제조사가 개별 항목으로 표시
- **커스텀 모델 내보내기/가져오기** — 제조사+모델 데이터를 JSON으로 export/import
- **기본 프롬프트 수정·내보내기** — 기본 프롬프트도 편집 및 내보내기 가능
- **README 재작성 및 CC BY-NC-SA 4.0 라이선스 추가**
- **채팅방별 모델 설정 저장 및 복원** - 채팅방마다 개별 모델 설정 유지
- **자동 요약 모델 선택 UI 개선** - 제조사 + 모델 2단 드롭다운으로 변경
- **요약 프롬프트 내보내기·가져오기** - 요약 프롬프트 설정 공유 기능
- **프롬프트 조건/프리셋 내보내기·가져오기** - 설정집 공유 확장
- **Google Vertex AI 모델 지원** - Vertex AI 채팅 모델 추가
- **설정 > 통계 스크린** - 사용 통계 확인 기능
- **프롬프트·사용자 메시지 주석 기능** - `/* */` 주석으로 내용 비활성화
- **채팅 드로어 프롬프트 프리셋 선택** - 드로어에서 프리셋·조건 편집 지원
- **프롬프트 항목 조건부 활성화** - 조건에 따라 프롬프트 항목 활성화
- **채팅 스크롤 이동 버튼** - 맨 위/맨 아래 이동 버튼 추가
- **커스텀 모델 API 키 직접 입력 지원**
- **API 로그 7일 자동 삭제**
- **캐릭터 태그 카드 표시**
- **메시지 원본 저장 및 렌더링 파이프라인 개선**
- **채팅 리스트 마지막 메시지 미리보기** - 표시 가능한 마지막 메시지 표시
- **로그에 모델 displayName 저장 및 표시**

### 개선
- 시작메시지를 첫 유저 메시지 바로 앞 model 턴으로만 삽입
- 시작설정/시작메시지 표시 시 {{user}} {{char}} 플레이스홀더 치환 적용
- 파일 가져오기 시 모든 확장자 선택 가능하도록 변경
- 자동 요약 기본값 조정 - 메시지 수 기준 10개, 파라미터 초기값 설정
- 자동 요약 기본 프롬프트 영문화
- 요약 프롬프트 기본값 업데이트 및 초기화 기능 추가
- 채팅 프롬프트 순서 보정 및 기본 프롬프트 초기화 기능 개선
- 메타데이터 태그 괄호 `[]` → `【】` 변경
- 드로어 제조사 선택 드롭다운 추가 및 레이아웃 개선
- 앱 표시 이름 `flan` → `Flan` 변경
- 채팅 목록 로딩 성능 개선 및 중복 코드 제거
- CommonFab·CommonCharacterCard 공통 위젯 추출
- 채팅 모델 설정 드롭다운 UI 간소화

### 버그 수정
- 캐릭터 카드 태그 다수 사용 시 bottom overflow 수정
- Agent 아이콘 위치 조정
- CommonDropdownButton에서 삭제된 항목 선택 시 AssertionError 수정
- CharacterScreen에서 dispose 후 setState 호출 오류 수정
- 기본 프롬프트 변수명 오타 수정 (ouput_language → output_language)
- 프롬프트 조건 변수 치환 시 중첩 변수 미해결 문제 수정
- Google AI Studio·Anthropic API 키 저장 시 검증 스킵 (비용 발생 방지)
- 상태창 캐릭터 이름 글자 크기가 뷰어 설정과 불일치하던 문제 수정
- 채팅 프롬프트 조건 ID 리매핑 누락 수정
- 자동 요약 모델 드롭다운 중복 value 에러 수정
- 커스텀 모델 Base URL `/v1` 자동 제거 로직 제거
- 시작설정/시작메시지 `{{}}` 키워드 치환 문제 수정
- 치환 키워드 값 내부 주석이 제거되지 않던 버그 수정

## v1.0.3

### 새로운 기능
- **캐릭터 카드 V2/V3(RisuAI) 가져오기 지원** - 이미지, 시작설정, 추가필드 포함
- **자동 요약 기능** - 핀 기반 자동 요약 생성, 프롬프트 커스터마이징, 재생성 및 수동 요약 추가
- **자동 요약 프롬프트 키워드 치환** - 요약 프롬프트에서 `{{char}}`, `{{chat_historys}}` 등 키워드 사용 가능
- **백업 및 복구 기능** - 전체 데이터 백업/복원 지원
- **설정집 가져오기/내보내기** - 프롬프트 설정 공유 기능
- **프롬프트 정규식 규칙 시스템** - 정규식 기반 프롬프트 변환 규칙
- **기본 프롬프트 시스템** - 기본 프롬프트 템플릿 제공
- **API 키 다중 등록 및 선택** - 여러 API 키 등록 후 라디오 버튼으로 선택
- **API 키 등록 시 검증 루틴** - 유효하지 않은 키 사전 차단
- **최신 AI 모델 추가** - OpenAI o3/o4-mini/GPT-4.1, Claude 4.5/4.6, Gemini 3.1 Pro Preview
- **커스텀 모델 및 테마 색상 지원**
- **채팅 핀 기능** - 메시지 핀, 자동/수동 핀 모드, 자동 핀 세부 옵션(날짜/장소/AI 자동)
- **메시지 핀 감지 시 Scene 태그 자동 삽입**
- **채팅방 우측 Drawer 메뉴** - 기본 정보/페르소나/캐릭터/로어북/요약 탭
- **채팅창 하단 설정 패널(케밥 메뉴)** 및 뷰어 설정
- **채팅 전송 시 2단계 로딩 인디케이터**
- **프롬프트 내보내기 폴더 구조 지원**

## v1.0.0 - 프로모션 첫 단계 릴리즈

### 주요 기능

#### 캐릭터 관리
- 캐릭터 생성, 수정, 삭제 기능
- 캐릭터 표지 이미지 설정
- 캐릭터 설명, 태그 관리
- 캐릭터 목록 그리드/리스트 뷰 전환
- 캐릭터 정렬 기능

#### 캐릭터 북 (로어북)
- 캐릭터 북 폴더 및 항목 관리
- 활성화 조건 설정 (활성화/비활성화/키워드)
- 키워드 기반 활성화 (AND/OR 조건)
- 삽입 순서 설정

#### 페르소나
- 사용자 페르소나 생성 및 관리
- 채팅 시 페르소나 선택 기능

#### 시작 시나리오
- 시작 설정(장면 묘사) 작성
- 시작 메시지(첫인사) 작성
- 여러 시나리오 생성 및 선택

#### 채팅 기능
- AI 캐릭터와 실시간 채팅
- 메시지 수정 및 삭제
- 메시지 재생성 기능
- 연속 메시지 자동 병합
- 채팅방별 토큰 카운트 표시
- 채팅 히스토리 자동 스크롤

#### 프롬프트 템플릿
- 커스텀 프롬프트 템플릿 생성
- 시스템/사용자/캐릭터/채팅 역할 지원
- 프롬프트 항목 드래그 앤 드롭 정렬
- 폴더 구조로 프롬프트 그룹화
- **키워드 치환 시스템**
  - `{{char}}` - 캐릭터 이름
  - `{{char_description}}` - 캐릭터 설명
  - `{{user}}` - 페르소나 이름
  - `{{user_description}}` - 페르소나 설명
  - `{{character_book}}` - 활성화된 캐릭터 북
  - `{{start_setting}}` - 시작 설정
  - `{{start_message}}` - 시작 메시지
- **채팅 히스토리 필터링**
  - 기본 모드: 전체 채팅 히스토리
  - 고급 모드: 최근/중간/오래된 채팅 범위 지정

#### UI/UX
